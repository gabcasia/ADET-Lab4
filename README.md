# REST API Project

## Features
- User registration
- Login validation
- Get user
- Update user

## Server
- PHP REST API
- JSON storage

## Clients
- Python CLI client
- JavaScript Node.js client

## How to Run

### Start Server
cd server
php -S localhost:8000

### Run Python Client
cd client-python
python client.py

### Run JS Client
cd client-js
node client.js
